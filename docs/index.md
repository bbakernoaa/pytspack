# Welcome to the pytspack documentation!

`pytspack` is a high-performance Python wrapper for Robert J. Renka's TSPACK library for tension spline curve fitting.

## Key Features

- **Tension Splines**: Control the 'tightness' of the interpolating curve to avoid overshoots or oscillations.
- **High Performance**: Core routines are implemented in C for speed.
- **xarray Integration**: Easily interpolate multi-dimensional atmospheric or oceanic data along a vertical dimension.
- **Dask-Aware**: Supports lazy computation on large datasets.
